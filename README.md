\# Sales Performance Analysis Dashboard



\## 📌 Project Overview

This project analyzes Superstore sales data using SQL and Power BI to identify revenue trends, profit margins, and top-performing states and categories.



\## 🛠 Tools Used

\- SQL (MySQL)

\- Power BI

\- Excel (for dataset)



\## 📊 Key Features

\- KPI Metrics (Total Sales, Profit, Orders, Profit Margin)

\- Monthly Sales \& Profit Trend Analysis

\- Top 5 States by Revenue

\- Category \& Sub-Category Analysis

\- 15-Day Sales Forecast



\## 💡 Business Insights

\- Technology category generates highest revenue

\- California is the top-performing state

\- Sales peak during Q4 season

\- Standard shipping mode is most used



\## 📁 Files Included

\- Power BI Dashboard (.pbix)

\- SQL Query File

\- Dataset CSV
