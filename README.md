# 📊 Sales Performance Analysis Dashboard using SQL, Excel & Power BI

## 📌 Project Overview

This project analyzes retail sales data using **SQL, Microsoft Excel, and Power BI** to evaluate sales performance, profitability, customer behavior, and regional trends.

The objective is to transform raw sales data into meaningful business insights through data analysis and interactive dashboards that support strategic business decision-making.


# 🎯 Business Problem

Retail organizations generate large volumes of sales data but often struggle to answer key business questions such as:

* Which regions generate the highest sales?
* Which product categories are most profitable?
* Who are the highest-value customers?
* Which products are generating losses?
* How do sales and profits change over time?
* Which customer segments contribute the most revenue?

This project converts raw sales data into actionable business insights that help improve business performance.
