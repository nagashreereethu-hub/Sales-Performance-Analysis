Sales Performance Analysis

📌 Project Overview

This project analyzes retail sales data using SQL, Microsoft Excel, and Power BI to evaluate sales performance, profitability, customer behavior, and regional trends.

The project demonstrates how business data can be transformed into meaningful insights through data analysis and interactive dashboards to support strategic business decisions.

🎯 Business Problem

Retail organizations generate large amounts of sales data but often struggle to identify:

Best-performing products
Most profitable regions
High-value customers
Sales trends over time
Business opportunities

This project helps convert raw sales data into actionable business insights.
