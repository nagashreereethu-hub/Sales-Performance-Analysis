# 📊 Sales Performance Analysis Dashboard

## 📌 Project Overview

The Sales Performance Analysis Dashboard is a business analytics project developed using **SQL, Microsoft Excel, and Power BI** to analyze retail sales performance, profitability, customer behavior, and regional trends.

The project transforms raw transactional data into interactive dashboards and actionable business insights, enabling organizations to monitor key performance indicators (KPIs), evaluate business performance, and support data-driven decision-making.


## 🎯 Business Problem

Retail organizations generate large volumes of sales data but often struggle to answer critical business questions such as:

- Which regions generate the highest revenue?
- Which product categories are most profitable?
- Who are the highest-value customers?
- Which products are generating losses?
- How do sales and profits change over time?
- Which customer segments contribute the most revenue?

This project addresses these challenges by converting raw sales data into meaningful business insights through interactive reporting and visualization.
